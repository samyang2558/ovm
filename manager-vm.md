# 管理虚拟机

### 部署虚拟机

首先需要有上传的可用模板

进入项目中点击vapp，在虚拟机的列表处。点击“+”添加虚拟机。在弹出的创建虚拟机对话框中输入虚拟的名称，选择模板，直接点击确定或者点击下一页设置cpu内存磁盘大小等点击确定

![](/assets/火狐截图_2016-09-26T06-32-14.294Z.png)

此时的虚拟还处于未部署状态，点击虚拟机的功能按钮，选择部署



![](/assets/火狐截图_2016-09-26T06-55-22.329Z.png)
